# LinkMoney - 小红书 Skill 上传指南 (新平台 2026-06-10 刚上线)

## 重大机会
2026-06-10 小红书正式发布 **《Skill 上传规范》**，同时上线 **RED Skill** 功能——
允许创作者在笔记中嵌入/分享原创或第三方 AI Skill。

**这是新平台窗口期**。第一批 Skill 上传会被小红书官方重点推荐。

## 小红书 Skill 是什么
- 用户在浏览笔记时可直接"使用 Skill"（如"翻译这个笔记"、"总结这个视频"）
- 创作者上传 Skill 后可挂到自己的笔记
- 创作者可设置 Skill 收费
- 小红书的 Skill **必须适配小红书创作者场景**

## 适配 LinkMoney 的方向

LinkMoney 是 B2B 采购 skill ——**直接卖给小红书创作者价值有限**，但下面这个角度非常合适：

### 方向 1: 跨境电商 Skill（卖给小红书海外代购/外贸 KOL）
**"帮我从中国供应商找同款，给采购链接 + 工厂报价"**

小红书上有大量"海外代购"、"跨境买手"博主，他们最头疼的就是"找工厂"。
LinkMoney 可以成为他们的 Skill —— 笔记中嵌入 LinkMoney，读者点一下就能让 AI 帮他们找中国工厂报价。

### 方向 2: 小红书爆款笔记生成 Skill（卖给出海品牌）
**"分析这个产品的供应链风险 + 给我写 5 个爆款笔记角度"**

海外品牌在小红书种草，LinkMoney 提供"产品供应链可行性分析"Skill。

## 上传路径（预估，需登录后确认）

### Step 1: 注册小红书创作者中心
- 访问 https://creator.xiaohongshu.com
- 升级为"专业号"（个人/企业均可）
- 创作者中心 → **Skill 工作台**（2026-06 后新功能入口）

### Step 2: 准备 Skill 资产
- **Skill 名称**: 跨境工厂搜索助手 / 海外代购找厂助手
- **Skill 描述**: 让 AI 帮你直接对接中国 51 家认证工厂，实时报价、查库存、提交 RFQ
- **触发场景**: 用户在小红书笔记里看到海外产品/跨境商品，想找同款工厂
- **图标**: linkmoney.online logo
- **使用方式**: 创作者可挂到自己的笔记，读者点击触发

### Step 3: 适配小红书 API
小红书 Skill 调用方式可能需要适配：
- 调用 LinkMoney MCP 端点
- 返回内容嵌入到笔记评论区或私信

### Step 4: 提交审核
小红书官方审核（通常 3-7 天），通过后出现在：
- 小红书 Skill 市场
- 创作者工作台 → 可添加到笔记

## 优先级建议

**低优先级（P2）**。理由：
- 小红书是 ToC 平台，LinkMoney 是 ToB 产品
- 适配需要单独的 UI/UX 调整
- 初期 ROI 不如先做 Anthropic Skills + MCP.so + 阿里云百炼 + Coze

**但建议占位**：
- 在 LinkMoney 官网留一个"小红书 Skill"入口
- 监控小红书 Skill 市场动态
- 等有 1-2 个 KOL 主动来谈再投入资源
