**Tags**: b2b, china, sourcing, supply-chain, manufacturing, procurement, agent, MCP, AI

**Description (EN)**:
LinkMoney is the first MCP Skill that connects AI agents directly to real Chinese factory ERP systems — enabling live pricing, real-time inventory, and automated RFQ without stale Alibaba listings.

Unlike traditional B2B platforms built for humans, LinkMoney is designed for agent-to-agent commerce. When an overseas procurement agent asks "Find M8 304 stainless steel bolts, 50K pcs", LinkMoney returns 3-5 verified Chinese factories with LIVE pricing from their own MCP servers.

**Key Differentiators**:
- Real-time data from factory ERP systems (not 3-month-old cached listings)
- 51 verified Chinese suppliers, 140 products across 7 categories
- 13 public MCP tools covering full procurement lifecycle
- Hybrid architecture: live data when factories online, cached fallback when offline
- Open source, MIT license

**Pricing**: 3% success fee on closed deals. First 3 months free.
