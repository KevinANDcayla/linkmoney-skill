API Usage Examples

Standard MCP protocol. Works with Claude, ChatGPT, Coze, Qwen, and any MCP-compatible agent.

# 1. Register as a buyer (one-time, free) POST /mcp/register_buyer { "email": "buyer@acme.com", "country": "USA", "company": "Acme Industries" }
# 2. Find Chinese suppliers for fasteners POST /mcp/find_china_supplier { "category": "fastener", "spec": "DIN 933 M8", "cert_required": ["ISO9001"], "moq_max": 50000 } # → returns 8-15 ranked suppliers with trust_score, reviews, certifications
# 3. Get real-time pricing POST /mcp/get_pricing { "supplier_id": "sup_001", "quantity": 25000, "incoterm": "FOB" } # → returns tiered pricing, lead time, inventory
# 4. Submit RFQ — factory responds within 5 business days POST /mcp/submit_rfq { "supplier_id": "sup_001", "product": "Hex Bolt DIN 933 M8x40", "quantity": 25000, "delivery_port": "Los Angeles", "notes": "Need zinc plating, ISO9001 cert" }
